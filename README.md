# 轻记账 (LiteBook)

> 一个简单、优雅的个人记账应用

轻记账是一款专注于个人日常收支记录的移动应用，采用极简设计理念，让记账变得轻松自然。

## ✨ 特性

- 🎯 极简设计：专注于最核心的记账需求
- 📊 智能统计：自动生成收支报表，轻松了解消费习惯
- 🏷️ 标签管理：灵活的支出收入分类，支持自定义标签
- 📅 多维度查看：支持日、周、月、年度账单查看
- 📱 离线可用：本地优先，无需担心隐私泄露
- 🔄 数据同步：可选的云端同步，设备间无缝切换
- 📈 预算管理：设置预算目标，合理规划支出
- 🌙 深色模式：保护夜间使用者的眼睛
